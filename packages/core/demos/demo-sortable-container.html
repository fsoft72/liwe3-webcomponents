Customization with CSS Variables

Customize the appearance using CSS custom properties.

Available CSS Variables:

  • --sortable-gap - Gap between items (default: 8px)
  • --sortable-indicator-color - Drop indicator color (default: #667eea)
  • --sortable-indicator-radius - Drop indicator border radius (default: 4px)
  • --sortable-handle-size - Handle width/height (default: 24px)
  • --sortable-handle-bg - Handle background color
  • --sortable-handle-hover-bg - Handle hover background
  • --sortable-handle-color - Handle icon color
  • --sortable-handle-hover-color - Handle icon hover color
  • --sortable-handle-radius - Handle border radius (default: 4px)
