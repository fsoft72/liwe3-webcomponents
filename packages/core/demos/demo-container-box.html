A container with hover-activated menu in configurable positions

Features

  • Wraps any HTML content (divs, images, custom elements)
  • Shows menu button on hover (or always visible)
  • Configurable menu position (4 corners)
  • Integrates with PopoverMenu component
  • Clicks pass through to wrapped content
  • Customizable with CSS variables
  • TypeScript support with full type definitions

Usage Example

// Import the ContainerBox component
import { ContainerBoxElement } from '@liwe3/webcomponents';

// Get the container element
const box = document.querySelector('liwe3-container-box');

// Set menu items
box.setMenuItems([
  {
    label: 'Edit',
    onclick: () => console.log('Edit clicked')
  },
  {
    label: 'Delete',
    onclick: () => console.log('Delete clicked')
  },
  { label: '---sep' },
  {
    label: 'More',
    items: [
      {
        label: 'Share',
        onclick: () => console.log('Share clicked')
      },
      {
        label: 'Duplicate',
        onclick: () => console.log('Duplicate clicked')
      }
    ]
  }
]);

// Set menu position
box.setMenuPosition('top-right');

// Make menu always visible
box.setAlwaysShowMenu(true);
